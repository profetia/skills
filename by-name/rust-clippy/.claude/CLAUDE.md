This is a development repository for Rust Clippy. The default remote is `origin`, which points to my fork of the official Clippy repository. The `upstream` remote points to the official Clippy repository. The default branch is `master` and branches that are named `issue1234` are branches that I created to work on a specific issue. 

The configurations for Claude are stored in the `.claude` directory and is not ignored by Git. Please work with care when using git so that you do not accidentally commit changes to the `.claude` directory or have them deleted.

Prefer rebase instead of merge when updating branches.
